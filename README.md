### React search text - by provided url web site

#### Requirements

* ##### Installed NodeJs > 18
* ##### Install git

#### Instructions

1. Open terminal 
2. Download `git clone https://github.com/maximcoding/react-free-search.git`

#### Install Server

1. `cd url-scrap-api`
2. `npm i`
3. `npm start`

#### Install Client

1. `cd url-scrap-client`
2. `npm i`
3. `npm start`
4. open `http://localhost:3000/`
